📜 什么是清华简与《算表》？ / Background & Citations

“清华简”是指清华大学于2008年收藏的一批战国楚国竹简，共近2500支。而《算表》则是这批竹简中最为特殊、由21支竹简组成的数学文献，经碳14测定书写于约公元前305年。它不仅是我国发现最早的实用算具，更被吉尼斯世界纪录认证为“人类最早的十进制计算器”。

📐 算表物理算力与位值制逻辑 / Physical Scale & Positional Logic

作为数字化精确复原版，本交互矩阵严格遵循两千年前文物的真实物理极限。
① 物理天花板 [0.5 - 99.5]：《算表》的行列固定表头由“十位系列(90-10)”、“个位系列(9-1)”及“半数(0.5)”组成。乘法与求平方模式下，输入的因子不得大于 99.5 且小数部分必须是 0 或 5（即只支持整数或半数组件拆解），超出或不符者将触发极限防御弹窗。
② 位值制拆解规则与极限支持：战国楚人已完美掌握“十进制位值制”。为了保护大表核心矩阵并支持高阶计算，除法与开方的输入上限统一提升并锁死在 9900.25（即大表终极理论上限 99.5X 99.5）。在除法与开方模式中，只要输入数字不大于 9900.25，系统即全面支持运算。若最终计算得出的只是近似值（如除不尽的无限循环小数、非完全平方数开方等），本数字化系统将自动为您保留三位小数，以便用现代科学视角评估历史算力。

🧭 《算表》运筹操作指南 / How to Use the Matrix

古人使用《算表》时，会利用竹简上的丝线进行交叉定位。
1. 乘法与平方：在顶部横轴和右侧纵轴找到要计算的数字（若有十位和个位，则拆开看），从对应的表头分别向下拉出纵向和横向引线。引线在内部交汇处的数字之和，即为最终结果。
2. 除法与开方：除法是乘法的逆运算。已知除数（顶行）和被除数（大表总和），在对应的纵向光轴区域内，逆向凑出等于被除数（或按大表量程等比压缩）的格子组合，其横向联动锁定的右列数字即为商。
